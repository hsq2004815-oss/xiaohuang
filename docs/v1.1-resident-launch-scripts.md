# V1.1 Resident Launch Scripts & Modes

> **Status:** Jarvis 常驻模式 — 一键启动/停止/状态查看、常驻隐藏、连续会话
> **Date:** 2026-05-02

## 当前能力总览

| 版本 | 能力 |
|------|------|
| V1.1.1B | 一键启动 / status / stop / 防重复进程 / secrets 自动加载 / 日志拆分 |
| V1.1.1C | Resident Hidden — 平时隐藏，唤醒后弹出，回复后隐藏 |
| V1.1.1D | Conversation Session — 唤醒后可连续对话，后续不用重复喊"小黄" |

## 脚本

| 脚本 | 作用 |
|------|------|
| `scripts/start_stt_server.ps1` | 后台启动 STT server (`127.0.0.1:8766`)，自动检查是否已运行 |
| `scripts/start_overlay.ps1` | 后台启动 voice overlay，支持所有 CLI 参数 |
| `scripts/start_xiaohuang.ps1` | 一键启动：STT server → 等待 /health ready (最多 60s) → 启动 overlay |
| `scripts/status_xiaohuang.ps1` | 查看运行状态：/health 关键字段 + 进程列表 |
| `scripts/stop_xiaohuang.ps1` | 停止 overlay；`-StopSttServer` 同时停止 STT server |

## 参数说明

| 参数 | 作用 | 默认 |
|------|------|------|
| `-Device` | 麦克风设备 ID | 0 |
| `-EnableLlm` | 启用 DeepSeek 单句回复 | off |
| `-EnableTts` | 启用 edge-tts 语音播放 | off |
| `-Debug` | 打印详细日志 | off |
| `-ResidentHidden` | 启动后隐藏浮窗，只在唤醒后显示 | off |
| `-ConversationSession` | 唤醒后连续多轮对话，无需重复喊"小黄" | off |
| `-SessionTimeout` | 每轮后续输入等待秒数 | 30 |
| `-MaxSessionTurns` | 一次连续会话最多轮数 | 5 |

## 基本使用

```powershell
cd E:\Projects\xiaohuang

# 完整启动（LLM + TTS + debug）
.\scripts\start_xiaohuang.ps1 -Device 0 -EnableLlm -EnableTts -Debug

# 查看状态
.\scripts\status_xiaohuang.ps1

# 停止（保留 STT server）
.\scripts\stop_xiaohuang.ps1

# 停止全部
.\scripts\stop_xiaohuang.ps1 -StopSttServer
```

## V1.1.1C 常驻隐藏模式

平时窗口隐藏，后台持续监听唤醒词。听到"小黄"后窗口弹出，回复结束后自动隐藏。

```powershell
.\scripts\start_xiaohuang.ps1 -Device 0 -EnableLlm -EnableTts -Debug -ResidentHidden
```

- 这不是系统托盘。
- 这不是开机自启动。
- 要停止仍使用 `.\scripts\stop_xiaohuang.ps1 -StopSttServer`。

## V1.1.1D 连续会话模式

第一次仍需说"小黄"唤醒。唤醒后进入连续会话窗口，后续几句可以直接说，不用重复喊"小黄"。

**启动：**

```powershell
.\scripts\start_xiaohuang.ps1 -Device 0 -EnableLlm -EnableTts -Debug -ResidentHidden -ConversationSession -SessionTimeout 30 -MaxSessionTurns 5
```

**退出词：**
好了、没事了、不用了、退出、结束、休息吧、可以了、停止、闭嘴、先这样

**典型对话：**

```text
用户：小黄
小黄：我在
用户：你好
小黄：你好呀……
用户：那邓小平呢？
小黄：……
用户：好了
小黄：好的，我先待命。
```

- 这不是全双工。
- 不支持打断 TTS。
- 不会执行真实工具。
- 退出词不区分上下文，只要命令文本匹配退出词就会退出 session。

## 本地 Secrets 配置

推荐使用 `$env:USERPROFILE\.xiaohuang\secrets.ps1` 管理 API key：

```powershell
New-Item -ItemType Directory -Force "$env:USERPROFILE\.xiaohuang" | Out-Null
notepad "$env:USERPROFILE\.xiaohuang\secrets.ps1"
```

文件内容（**绝对不要提交到 Git**）：

```powershell
$env:DEEPSEEK_API_KEY="your_real_key_here"
```

- `secrets.ps1` 位于用户目录，不在 Git 仓库内
- `.gitignore` 已排除 `.env` / `secrets.ps1` / `.xiaohuang/`
- 启动脚本会优先使用当前 shell 的环境变量，未设置则自动加载 secrets
- 如果 Windows 系统环境变量中有旧 `DEEPSEEK_API_KEY`，可能会覆盖 secrets
