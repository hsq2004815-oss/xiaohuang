- 不接 OpenCLI / opencode / 浏览器自动化 / QQ / 微信 / 爬虫
- 不接个人数据库 / 长期记忆 / 多 Agent / 插件市场
- 不接微服务 / 复杂数据库
- 不拆分 overlay_controller / _run_overlay_loop
- 不修改 DeepSeek/TTS/fallback 内部策略
- 不写入 E:\DataBase

## 5. 安全边界

- 所有 API key 通过环境变量注入，不写入代码/配置/日志
- STT server 只监听 127.0.0.1
- `/transcribe` 路径 guard：只接受 `data/recordings/**/*.wav`
- last_error 不记录 traceback/路径/token
- build_deepseek_response_debug_summary 不输出 Authorization/prompt/完整响应
- task_router 不调用任何外部工具
